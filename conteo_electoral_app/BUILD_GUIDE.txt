# Flutter Build Configuration for Android APK

## Requisitos Previos
1. Instalar Flutter SDK (https://flutter.dev)
2. Android Studio con SDK de Android
3. Java Development Kit (JDK 11+)

## Comandos para Generar APK

### APK Debug (para pruebas)
```bash
cd conteo_electoral_app
flutter pub get
flutter build apk --debug
```

El APK se generará en: `build/app/outputs/flutter-apk/app-debug.apk`

### APK Release (para producción)
```bash
cd conteo_electoral_app
flutter build apk --release
```

El APK se generará en: `build/app/outputs/flutter-apk/app-release.apk`

### APK Split (por arquitectura)
```bash
cd conteo_electoral_app
flutter build apk --split-per-abi
```

Genera APKs separados para cada arquitectura:
- `app-armeabi-v7a-release.apk` (32-bit)
- `app-arm64-v8a-release.apk` (64-bit)
- `app-x86_64-release.apk` (emulador)

## Configuración de Firma (Release)

Para generar un APK release firmado:

1. **Crear keystore:**
```bash
keytool -genkey -v -keystore conteo-electoral.jks -keyalg RSA -keysize 2048 -validity 10000 -alias conteoelectoral
```

2. **Crear archivo `android/key.properties`:**
```properties
storePassword=TU_PASSWORD
keyPassword=TU_PASSWORD
keyAlias=conteoelectoral
storeFile=conteo-electoral.jks
```

3. **Modificar `android/app/build.gradle`:**
```groovy
def keystorePropertiesFile = rootProject.file("key.properties")
def keystoreProperties = new Properties()
keystoreProperties.load(new FileInputStream(keystorePropertiesFile))

android {
    signingConfigs {
        release {
            keyAlias keystoreProperties['keyAlias']
            keyPassword keystoreProperties['keyPassword']
            storeFile rootProject.file(keystoreProperties['storeFile'])
            storePassword keystoreProperties['storePassword']
        }
    }
    buildTypes {
        release {
            signingConfig signingConfigs.release
        }
    }
}
```

4. **Generar APK:**
```bash
flutter build apk --release
```

## Cambiar IP del Servidor

Para dispositivo físico, editar:
`lib/services/api_service.dart` línea 7:

```dart
// Emulador Android (localhost del host)
static const String baseUrl = 'http://10.0.2.2:8080/api';

// Dispositivo físico (tu IP local)
// static const String baseUrl = 'http://192.168.1.100:8080/api';
```

## Instalar en Dispositivo Conectado

```bash
flutter install
```

## Comandos Útiles

```bash
# Limpiar build anterior
flutter clean

# Ver dependencias
flutter pub deps

# Actualizar paquetes
flutter pub upgrade

# Generar bundle JS
flutter build web
```

## Estructura del Proyecto

```
conteo_electoral_app/
├── lib/
│   ├── main.dart              # Punto de entrada
│   ├── models/models.dart     # Modelos de datos

│   ├── services/             # API HTTP
│   ├── providers/            # Estado global

├── ios/                      # Config iOS
```

## Configuración Mínima Android

En `android/app/build.gradle`:
```groovy
android {
    defaultConfig {
        applicationId "com.electoral.conteoelectoral"
        minSdkVersion 21
        targetSdkVersion 34
    }
}
```

## Permissions Requeridas

En `android/app/src/main/AndroidManifest.xml`:
```xml
<uses-permission android:name="android.permission.INTERNET"/>
<uses-permission android:name="android.permission.ACCESS_NETWORK_STATE"/>
```
